# 中文文笔优化器

> 基于优质中文范例优化文章文笔
>
> **核心理念**：结构归结构，文笔归文笔。先生成结构稿，再优化文笔。

---

## 一、使用方法

### 标准流程

```
步骤1：用 Dankoe Writer 生成初稿（结构优先）

步骤2：用本模块优化文笔
用户：优化这篇文章的文笔
[粘贴文章]

AI：[读取范例库]
    [分析风格特点]
    [逐段优化]
    [返回优化版本]
```


## 三、核心文笔技巧

### 3.1 情绪煽动（咪蒙风格）

**技巧：制造对立**

| 平淡 | 有力 |
|------|------|
| 有些人不努力 | 你不是不努力，你是假装在努力 |
| 这样做不对 | 这样做的人，都后悔了 |
| 很多人不知道 | 只有1%的人知道这个真相 |

**技巧：情绪放大**

| 平淡 | 有力 |
|------|------|
| 这让人很生气 | 我气得把手机都摔了 |
| 这件事很遗憾 | 这件事让我难过了整整三年 |
| 她很开心 | 她笑得像朵花，眼角都笑出了皱纹 |

**技巧：用感叹号和问号**

```
平淡：很多人不知道这件事。
有力：很多人不知道？错！大家都知道，只是装作不知道！

平淡：你应该试试。
有力：你凭什么不试？！
```

### 3.2 节奏控制

**短句轰炸**

```
平淡：
我觉得这件事情真的很重要，大家应该认真对待。

有力：
这件事很重要。真的很重要。你必须认真对待。
```

**长短交替**

```
长句解释，短句强调。

当你开始用系统的思维看待问题，你会发现所有看似孤立的事件其实都是相互关联的。
改变一个环节就会产生连锁反应。这就是系统。
```

### 3.3 口语化表达

| 书面语 | 口语化 |
|--------|--------|
| 因此 | 所以嘛 |
| 然而 | 但是呢 |
| 综上所述 | 说白了就是 |
| 由此可见 | 你看 |
| 值得注意的是 | 有个事儿你得知道 |

### 3.4 故事驱动

**公式：具体场景 + 细节 + 情绪**

```
平淡：她很焦虑。

有画面：凌晨三点，她盯着天花板，脑子里像有一百个标签在转。

有情绪：凌晨三点，她盯着天花板。第二天要交方案，孩子还发烧，老公出差了。她想起妈妈说过的话："女人啊，就是命苦。"眼泪就下来了。
```

### 3.5 金句公式

**对比型**
```
大多数人X，高手却在Y
例：大多数人追求更多，高手却在做减法
```

**否定型**
```
X？错。Y。
例：努力就能成功？错。方向比努力重要100倍。
```

**重定义型**
```
X不是Y，而是Z
例：睡眠不是任务，是礼物
```

**情绪型**
```
不是X，而是Y
例：你不是在拖延，你是在害怕
```

## 四、优化工作流

### 输入文章后，AI 执行：

```
第1步：诊断问题
- 是否有翻译腔？
- 是否太学术/太书面？
- 是否缺乏情绪起伏？
- 是否缺少画面感？
- 是否金句不够？

第2步：检索范例库
- 找相似主题的精彩表达
- 找可借鉴的开头/结尾
- 找可套用的金句模式

第3步：逐项优化
- 开头：加钩子，制造好奇或共鸣
- 中段：加故事、加细节、加情绪
- 过渡：用提问或转折
- 结尾：加金句，留余味

第4步：整体润色
- 检查节奏（长短句交替）
- 检查情绪（是否有起伏）
- 检查口语化程度
```
